# Supplementary Materials

This folder contains additional materials to support the publication submission.

## Contents

1. **System Architecture Diagram** (`architecture_diagram.md`) - Visual representation of the multi-agent system architecture
2. **Workflow Diagram** (`workflow_diagram.md`) - LangGraph workflow visualization
3. **Example Output** (`example_output.md`) - Sample analysis report from the system
4. **Agent Interaction Flow** (`agent_interaction.md`) - Detailed agent communication patterns
5. **Tool Specifications** (`tool_specifications.md`) - Detailed tool documentation
6. **Installation Guide** (`installation_guide.md`) - Step-by-step installation instructions
7. **Quick Start Guide** (`quick_start.md`) - Quick reference for using the system

## Usage

These materials can be uploaded to Ready Tensor as supplementary files to provide additional context and documentation for reviewers.
