# The Multilingual Mandi

A web platform for local vendors that provides instant AI-driven price discovery and negotiation tools.

## Project Overview

This project is built using Kiro and includes real-time linguistic bridge capabilities for local trade.

## Setup

1. Clone this repository
2. Install dependencies
3. Run the development server

## GitHub Repository Setup

### Quick Setup (Recommended)

1. **Get a GitHub Personal Access Token:**
   - Go to: https://github.com/settings/tokens/new
   - Select "repo" scope
   - Generate and copy the token

2. **Run the setup script:**
   ```powershell
   $env:GITHUB_TOKEN = 'your-token-here'
   .\create-repo.ps1 -RepoName "multilingual-mandi"
   ```

This will:
- Create the GitHub repository
- Push your code
- Enable GitHub Pages automatically

### Manual Setup

See [GITHUB_SETUP.md](GITHUB_SETUP.md) for detailed manual instructions.
