# Data Weaver Project Guidelines

## Project Overview
This project demonstrates the power of combining unrelated data sources to discover interesting patterns and correlations. We're building a dashboard that merges weather data with stock market information.

## Development Standards

### Code Quality
- Use modern JavaScript/React patterns
- Implement proper error handling
- Add loading states for better UX
- Follow responsive design principles

### API Integration
- Weather data from OpenWeatherMap API
- Stock data from Alpha Vantage API
- Implement proper rate limiting
- Cache responses when appropriate

### Data Visualization
- Use Chart.js for interactive charts
- Implement correlation analysis
- Show both real-time and historical data
- Make insights easily digestible

### Kiro Acceleration Points
- Rapid API integration setup
- Component scaffolding
- Error handling implementation
- Responsive design patterns
- Documentation generation
