# Smart Desktop Organizer - Project Context

## Project Overview
This is a "Lazy Automation" project for Kiro Week 2 Challenge that solves the boring task of manually organizing cluttered directories.

## Development Approach
- Focus on practical utility over complexity
- Use only Python standard library for maximum compatibility
- Provide both dry-run and actual execution modes
- Include comprehensive error handling and user feedback

## Key Features to Maintain
- Cross-platform compatibility (Windows, macOS, Linux)
- Smart file type detection with 50+ extensions
- Date-based organization (YYYY-MM format)
- Duplicate file handling with automatic renaming
- Detailed reporting and statistics

## Code Quality Standards
- Clear, readable Python code with proper docstrings
- Comprehensive error handling for file operations
- User-friendly command line interface
- Informative progress feedback and summaries

## Testing Considerations
- Test with various file types and sizes
- Verify cross-platform path handling
- Test duplicate file scenarios
- Validate date-based folder creation
