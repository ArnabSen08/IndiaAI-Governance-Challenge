# Leonardo.ai Image Generation Prompt

## Prompt for Data Weaver Dashboard Cover Image

**Main Prompt:**
```
A modern, sleek dashboard interface showing weather and stock market data visualization, split-screen design with weather icons (sun, clouds, rain) on the left side and financial charts with green and red candlesticks on the right side, connecting lines and data streams flowing between them showing correlation, futuristic holographic display style, blue and green color scheme, professional business aesthetic, high-tech data visualization, 3D elements, clean minimalist design, digital art style
```

**Style Settings:**
- Style: Digital Art / Concept Art
- Aspect Ratio: 16:9 (for blog cover image)
- Quality: High
- Creativity: Medium-High

**Additional Keywords:**
```
dashboard, data visualization, weather icons, stock charts, correlation analysis, modern UI, holographic display, financial technology, data science, business intelligence, clean design, professional, futuristic interface
```

**Negative Prompt:**
```
cluttered, messy, low quality, blurry, dark, outdated design, cartoon style, childish, unprofessional, too many colors, chaotic layout
```

## Alternative Prompt (More Abstract):

**Abstract Data Flow Prompt:**
```
Abstract representation of data flowing between weather symbols and financial graphs, geometric shapes representing data points, flowing lines connecting weather patterns to market trends, modern gradient background from blue to green, minimalist design, professional business style, digital transformation concept, clean vector art style, high contrast, sophisticated color palette
```

## Usage Instructions:

1. Copy the main prompt into Leonardo.ai
2. Set the style to "Digital Art" or "Concept Art"
3. Choose 16:9 aspect ratio for blog cover
4. Generate 4 variations
5. Select the best one that shows clear connection between weather and financial data
6. Use for your AWS Builder Center blog post cover image

## Tips:
- The image should clearly represent the concept of connecting weather data with financial markets
- Avoid overly complex designs that might be hard to read as a cover image
- Ensure the image looks professional and suitable for a technical blog post
- The 16:9 aspect ratio works best for blog headers and social media sharing
